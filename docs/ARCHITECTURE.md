# INTROBOT — Architecture

> Find warm-intro paths through your team's combined network graph and draft double-opt-in intro requests from a single contacts manifest.

```
input ──▶ collect ──▶ rules/analyzers ──▶ score ──▶ findings ──▶ table · json
                              │                          │
                         (this repo)                 MCP tool (agents)
```

- **collect** normalizes the target (file/dir/API) into records.
- **rules/analyzers** apply the heuristics shipped in `introbot/core.py`.
- **score** ranks by severity.
- **MCP server** (`introbot mcp`) exposes `scan` for Cognis.Studio agents.

Extend by adding a rule + a test + a `demos/NN-*/SCENARIO.md`. See [CONTRIBUTING.md](../CONTRIBUTING.md).
